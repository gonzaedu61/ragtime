Here is an English summary of the main master-data topic areas handled in ALMENDRO (all names refer to the corresponding PDF/workshop where you find the details):

1. Address master data (stamm_adr.pdf)  
2. “Other” master data (stamm_sonst.pdf / Workshop “Stammdaten Sonstige”)  
3. Material master data (Workshop “Stammdaten Material”)  
4. Calculation master data (stamm_kalk.pdf / Workshop “Stammdaten Kostenstellen, Kalkulation”):  
   - Cost centers  
   - Operations (Arbeitsgänge)  
   - Service/performance catalogs (Leistungskatalog)  
5. Text tables (short & long):
   - 1–4-digit key + 20- or 50-character description
   - From v10.1 you can block individual entries so they no longer appear in pick lists
6. Freight tables (“Standard” and “Lieferant” versions) for tariff calculation in costing and delivery notes
7. Seller (mandant) payment-receipt data (in the seller’s master data)
8. Reference quantities (Bezugsgrößen), e.g. for interpolation or operation quantities (handled by ALMENDRO support)
9. Control parameters for the standard cost calculation result (adm_kerg.pdf)
10. GDPR-driven anonymization settings for personal data
11. Data-import/-export definitions (e.g. large-volume Excel upload/download of master data)
